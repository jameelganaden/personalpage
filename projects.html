Twitch User Engagement Prediction Model

[09/2024 - 11/2024]

Created and optimized a predictive model to forecast Twitch user engagement...

I led the creation of a machine learning model that predicted Twitch user engagement using Python, Scikit-Learn, and Random Forest Regression. Through data analysis, we identified key engagement metrics, such as watch time and session durations, which improved our predictions. I facilitated the project, ensuring the team developed simple and complex models while maintaining data integrity. The final model achieved an MAE of 2.08, the best performance among several evaluated models.
